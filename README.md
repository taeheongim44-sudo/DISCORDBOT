# Trickcal Discord Bot

디스코드 서버용 자동 공지/쿠폰 알림 봇입니다.

## 🚀 기능
- `!공지 업데이트` : 최신 업데이트 공지 가져오기  
- `!공지 쿠폰` : 최신 쿠폰 공지 가져오기  
- `!명령어` : 명령어 목록 보기  
- 자동으로 새 공지/쿠폰 감지 후 알림 발송  
- Render 24시간 유지 서버 포함  

## 🛠️ 설치 방법
1. 이 프로젝트를 GitHub에 업로드
2. Render(https://render.com) 접속 → **New Web Service**
3. GitHub 연결 → 이 프로젝트 선택
4. Build Command: `npm install`
5. Start Command: `npm start`
6. Environment Variables → `TOKEN` 추가 (디스코드 봇 토큰)

## ✅ 완성!
배포 후 Render URL을 방문하면  
`✅ Trickcal Discord Bot Running` 메시지가 나오면 정상 실행입니다.
